Frontend placeholder

This folder contains a minimal placeholder to indicate where an Angular frontend could be implemented.
The backend exposes a REST API that can be consumed by any frontend framework (Angular, React, etc.).

To implement an Angular frontend:
    1. Run `ng new frontend` (requires Node and Angular CLI)
    2. Implement a service that POSTs to /api/analyze and displays results.

This placeholder does not need to be executed to run the backend.
